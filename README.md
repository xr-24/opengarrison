## WIP

### How to Play

1. Download latest `.zip` or `.tar` from Releases
2. Extract somewhere
3. Run GG2.Client from that directory (linux: `sh run-client.sh`)
4. To host: Run GG2.ServerLauncher

### How to Do Stuff

1. Clone this repo
2. Make changes
3. Build with `dotnet build "C:/path/to/cloned/repo/GG2.sln" -c Release`
4. Distribute using instructions in GG2.CSharp/packaging/DISTRO QUICKSTART.txt
