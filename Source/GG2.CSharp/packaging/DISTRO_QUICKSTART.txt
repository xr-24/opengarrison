how to build

from `Source\GG2.CSharp`:

windows:
powershell -ExecutionPolicy Bypass -File .\scripts\package.ps1

linux/macOS:
pwsh ./scripts/package.ps1

build to windows only:
powershell -ExecutionPolicy Bypass -File .\scripts\package.ps1 -Platforms @('win-x64')

build to linux only:
pwsh ./scripts/package.ps1 -Platforms @('linux-x64')

build to macOS only:
pwsh ./scripts/package.ps1 -Platforms @('osx-x64','osx-arm64')

output:
Source\GG2.CSharp\dist\opengarrison-*

notes:
- the script runs dotnet test GG2.sln -c Release first
- it publishes GG2.Client, GG2.ServerLauncher, and GG2.Server, then copies assets/configs and archives them
